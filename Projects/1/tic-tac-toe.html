#1 TIC TAC TOE

Skills: JS, Arrays, HTML DOM

This project creates tic tac toe in HTML, CSS, and JavaScript using each for structure, styling, and functionality, respectively. On each click of the grid cells, the function runGame() is called. This works by first placing a marker, setting the active player, checking for a win or draw, and finally switching the player.

A predefined array is initialized with 9 string values of "n", where clicking a cell of a numbered id changes the array's corresponding value to "x" or "o". The checkWin() function checks for the array elements with x's or o's.
